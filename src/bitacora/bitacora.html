🛠️ Dificultades Encontradas y Soluciones

Responsive Design

Desafío: Lograr que el diseño se adapte correctamente a los tres breakpoints requeridos (400px, 900px, 1200px) sin que se rompiera la estructura.

Solución: Implementamos un sistema flexible con CSS y utilizmos media queries específicas para cada breakpoint. Probamos el diseño en múltiples dispositivos (simulados en el navegador) para asegurar la consistencia.
